Project: Scalable Time-Series Forecasting Engine
Objective: Optimized sales forecasting using a hybrid approach of Statistical (ARIMA/SARIMAX), Gradient Boosting (LightGBM), and Deep Learning (NeuralProphet) models.
Engineering Excellence: Implemented a modular architecture using custom source modules (src) for data ingestion and feature engineering, ensuring 100% code reusability.
Key Features: Integrated USFederalHolidayCalendar for holiday-aware forecasting and used TimeSeriesSplit for robust cross-validation.
Production Ready: Models are serialized using joblib for instant deployment and inference.
